# Etch-A-Sketch

A simple Etch-A-Sketch project to practice my DOM manipulation skills.

## Built with
- HTML
- CSS
- JavaScript

## How to play this
To play this, just hover over the grids to draw and if you want to change the grid size you can press the change grid button then an alert box will appear at the top, you are not allowed to enter integers above 100 or less than 1 and characters and decimals are also not allowed. If you want the color of the grids to be different, just select the RGB color button.
